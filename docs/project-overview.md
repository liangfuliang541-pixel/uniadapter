# 心迹 (XinJi) - 情绪追踪日记应用

## 项目概述

**心迹** (XinJi) 是一款专注于情绪追踪和日记记录的应用，旨在帮助用户更好地理解和管理自己的情绪状态。通过记录每日心情、撰写日记、分析情绪模式，用户可以发现情绪的变化规律，从而提升自我认知和心理健康水平。

## 核心功能

### 1. 心情追踪
- **快速签到**：通过5个等级的表情符号快速记录心情状态
- **心情分数**：将心情量化为1-5分，便于统计分析
- **日期关联**：自动关联心情与具体日期

### 2. 日记记录
- **富文本编辑**：支持多种格式的日记内容记录
- **照片附件**：可添加图片丰富记录内容
- **标签系统**：为日记添加标签便于分类检索
- **情绪因素**：记录影响心情的具体因素

### 3. 数据可视化
- **心情日历**：以日历形式展示历史心情变化
- **统计图表**：展示心情分布、趋势分析
- **因素分析**：分析不同因素对心情的影响程度
- **成就系统**：通过徽章激励用户持续记录

### 4. 智能功能
- **心情语录**：根据心情状态推荐治愈语录
- **写作提示**：提供引导式写作建议
- **数据分析**：智能分析情绪模式和趋势

## 技术架构

### 前端技术栈
- **React 18**：现代前端框架，提供优秀的用户体验
- **TypeScript**：强类型系统，提高代码质量和可维护性
- **Tailwind CSS**：实用优先的CSS框架，快速构建美观界面
- **Vite**：现代化构建工具，提供快速的开发体验
- **React Router**：声明式路由管理

### 设计系统
- **色彩系统**：基于HSL的颜色体系，支持动态主题切换
- **组件库**：基于原子设计理念的可复用组件
- **动画系统**：流畅的过渡动画提升用户体验
- **响应式设计**：移动端优先的响应式布局

### 架构特点
- **多平台适配**：采用适配器模式，支持H5、小程序、APP
- **业务分离**：核心业务逻辑与平台实现完全解耦
- **状态管理**：轻量级状态管理方案
- **数据持久化**：本地存储 + 可选云端同步

## 特色功能

### 1. 情绪因素分析
通过记录影响心情的因素（工作、健康、社交、爱好等），应用会分析这些因素对心情的影响程度，帮助用户更好地了解自己的情绪模式。

### 2. 智能语录推荐
根据用户当前的心情状态，智能推荐相应的治愈语录，提供情感支持和心理慰藉。

### 3. 成就激励系统
设计了一系列成就徽章，通过游戏化的方式激励用户持续记录，培养良好的情绪管理习惯。

### 4. 数据导出功能
支持将日记数据导出为JSON或Markdown格式，方便用户备份或迁移数据。
