# 🚀 UniAdapter - 智能多端适配器框架

## 项目概述

我们成功创建了一个名为 **UniAdapter** 的智能多端适配器框架，这是一个创新性的解决方案，旨在解决前端开发者在多平台开发中遇到的重复劳动和平台差异问题。

## 🏗️ 核心架构

### 1. 技术架构
- **核心适配器模式**: 通过统一接口屏蔽不同平台的实现差异
- **智能平台检测**: 自动识别运行环境并选择最优适配方案
- **统一API设计**: 一套代码适配所有平台

### 2. 文件结构
```
src/
├── core/                      # 核心架构层
│   ├── platform-detector.ts   # 平台检测器
│   └── adapter.ts            # 抽象适配器基类
├── hooks/                     # React统一Hooks
│   ├── usePlatform.ts        # 平台检测Hook
│   ├── useUniState.ts        # 统一状态管理
│   ├── useUniRouter.ts       # 统一路由适配
│   └── useUniRequest.ts      # 统一网络请求
├── platforms/                 # 平台特定实现
│   └── storage.ts            # 统一存储适配器
└── components/               # 可复用UI组件
    └── MultiPlatformDemo.tsx # 多平台展示演示
```

## 🎨 核心功能

### 1. 统一API接口
- `useUniState`: 统一状态管理，自动处理不同平台的持久化策略
- `useUniRouter`: 统一路由管理，提供跨平台的导航功能
- `useUniRequest`: 统一网络请求，屏蔽平台差异的HTTP请求
- `usePlatform`: 平台检测，获取当前运行环境信息

### 2. 智能适配策略
根据不同平台自动选择最优的技术方案：
- Web端: localStorage, History API, fetch
- 小程序端: 智能适配小程序API
- App端: 原生API适配
