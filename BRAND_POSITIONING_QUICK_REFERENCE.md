## 💬 常见问题速答

### Q: 什么是 UniAdapter？
```
A: 给你现有项目加多端能力的开源工具。
   只需 npm install，5 分钟集成。
```

### Q: 和 uniapp 什么关系？
```
A: 没有关系。完全不同的产品。
   uniapp 是框架（从零开始），我们是工具（现有项目）。
```

### Q: 为什么不是框架？
```
A: 框架会改变你的项目结构和开发方式。
   我们只是让你用现有工具和框架支持多个平台。
```

### Q: 能替代 uniapp 吗？
```
A: 不能，而且我们不想替代。
   我们服务的用户群体完全不同。
```

### Q: 学习成本高吗？
```
A: 零。useUniRouter、useUniRequest、usePlatform，就这三个。
   普通的 React/Vue Hook，5 分钟上手。
```

### Q: 为什么只有 2KB？
```
A: 因为我们只做适配，不做框架。
   所有其他功能（路由、状态管理等）用你现有的就行。
```
