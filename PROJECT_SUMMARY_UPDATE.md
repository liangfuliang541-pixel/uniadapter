# 🚀 UniAdapter - 智能多端适配器框架（Go分布式系统增强版）

## 项目概述

我们成功创建了一个名为 **UniAdapter** 的智能多端适配器框架，这是一个创新性的解决方案，旨在解决前端开发者在多平台开发中遇到的重复劳动和平台差异问题。最新版本已支持Go分布式系统，实现从前端到后端微服务的统一适配。

## 🏗️ 核心架构增强

### 1. 技术架构扩展
- **核心适配器模式**: 通过统一接口屏蔽不同平台的实现差异
- **智能平台检测**: 自动识别运行环境并选择最优适配方案
- **统一API设计**: 一套代码适配所有平台（包括Go分布式系统）
- **Go分布式适配器**: 支持微服务、RPC、消息队列、服务发现等

### 2. 文件结构更新
```
src/
├── core/                           # 核心架构层
│   ├── platform-detector.ts        # 平台检测器

├── hooks/                          # React统一Hooks
```

## 🎨 核心功能扩展

### 1. 统一API接口（新增Go分布式支持）
- `useUniState`: 统一状态管理，自动处理不同平台的持久化策略
- `useUniRouter`: 统一路由管理，提供跨平台的导航功能
- `useUniRequest`: 统一网络请求，屏蔽平台差异的HTTP请求
- `usePlatform`: 平台检测，获取当前运行环境信息
- `GoDistributedRPCAdapter`: Go分布式系统RPC调用
- `GoDistributedMessageQueueAdapter`: Go分布式消息队列
- `GoDistributedServiceDiscoveryAdapter`: Go分布式服务发现
- `GoDistributedLockAdapter`: Go分布式锁协调

### 2. 智能适配策略（新增Go分布式）
根据不同平台自动选择最优的技术方案：
- Web端: localStorage, History API, fetch
- 小程序端: 智能适配小程序API
- App端: 原生API适配
- **Go分布式端**: 微服务、RPC、消息队列、服务发现
